🔨 O que é Hammerspoon?

Hammerspoon é uma ferramenta poderosa de automação para macOS que permite criar atalhos personalizados, automatizar tarefas e controlar seu sistema usando scripts em Lua. É como ter superpoderes no seu Mac! 🚀

⚡ Automação Poderosa

Crie atalhos para qualquer ação: abrir apps, controlar janelas, executar scripts

🎯 Totalmente Customizável

Configure atalhos únicos que se adaptam ao seu workflow específico

🔓 Código Aberto

Gratuito, seguro e com uma comunidade ativa de desenvolvedores

📥 Como Instalar o Hammerspoon

🍺 Via Homebrew (Recomendado) brew install --cask hammerspoon
🌐 Download Direto

hammerspoon.org

🚀 Atalhos Básicos

📂 Abrir Arquivos/URLs ✅ Funciona
⌘ I
Abre URLs, caminhos de arquivo ou itens selecionados no Finder. Funciona universalmente em qualquer aplicativo.
Exemplo: Selecione "https://github.com" e pressione ⌘ I para abrir no navegador
📱 Mission Control ✅ Funciona
⌘ J
Mostra todos os Desktops e janelas abertas, permitindo navegação rápida entre espaços de trabalho.
Uso: Pressione ⌘ J para ver todos os seus desktops e janelas

🛠️ Ferramentas do Sistema

🎨 Digital Color Meter ✅ Funciona
⌘ ⌥ ⌃ T
Abre o seletor de cores nativo do macOS para capturar cores da tela.
Dica: Use ⌘+L no Color Meter para copiar a cor
💀 Force Quit ⚠️ Pode não funcionar
⌘ ⌥ ⌃ Q
Abre a janela Force Quit Applications para fechar aplicativos travados.
Alternativa: Use ⌘ ⌥ Esc se não funcionar
📊 Activity Monitor ✅ Funciona
⌘ ⌥ ⌃ A
Abre o Monitor de Atividade para verificar processos, CPU, memória e performance.
Uso: Monitore performance e processos do sistema
🔐 Passwords ✅ Funciona
⌘ ⌥ ⌃ P
Abre o gerenciador de senhas nativo do macOS para acessar suas credenciais.
Uso: Acesso rápido às suas senhas salvas
⌨️ Text Replacement ✅ Funciona
⌘ ⌥ ⌃ R
Abre diretamente a seção Text Replacement nas Configurações do Sistema para gerenciar atalhos de texto.
Uso: Configure expansões de texto como "omw" → "On my way!"

🔍 OCR - Reconhecimento de Texto

🔍 OCR de Área da Tela ✅ Funciona
⇧ ⌃ ⌘ R
Captura uma área selecionada da tela e extrai todo o texto visível usando OCR. Perfeito para PDFs escaneados, imagens com texto, vídeos pausados ou qualquer conteúdo não-selecionável.
Como usar:
1. Pressione ⇧⌃⌘R
2. Selecione área da tela com texto
3. Texto é copiado automaticamente
Requer: Permissões de Gravação de Tela
📁 OCR de Arquivo ✅ Funciona
⇧ ⌃ ⌘ F
Extrai texto de uma imagem selecionada no Finder. Suporta PNG, JPG, GIF, TIFF e outros formatos. Ideal para screenshots salvos ou fotos com texto.
Como usar:
1. Selecione imagem no Finder
2. Pressione ⇧⌃⌘F
3. Texto é extraído e copiado
Tecnologia: Tesseract + Vision API

🛠️ Configuração OCR

Dependências: Tesseract instalado via Homebrew

brew install tesseract tesseract-lang

Permissões: Configurações → Privacidade → Gravação de Tela → Adicionar Hammerspoon

Idiomas: Português + Inglês (configurável)

🖥️ Controle de Interface

🖥️ Show Desktop ⚠️ Pode não funcionar
⌘ ⌥ ⌃ Space
Esconde ou mostra todas as janelas, revelando o desktop. Tenta múltiplas estratégias (F11, fn+F11, hideOthers).
Alternativa: Use F11 ou fn+F11 se não funcionar
📋 Copiar Caminho do Finder ✅ Funciona
⌘ ⇧ W
Copia o caminho completo do arquivo/pasta selecionado no Finder. Se nada estiver selecionado, copia o caminho da pasta atual.
Exemplo: Selecione "projeto.txt" → ⌘ ⇧ W → copia "/Users/nome/Documents/projeto.txt"

🔗 Encurtador de URLs

🔗 TinyURL ✅ Funciona
⌘ ⇧ U
Encurta a URL selecionada usando o serviço TinyURL e copia o resultado para a área de transferência.
Como usar: Selecione uma URL e pressione ⌘ ⇧ U
📱 TinyURL + QR Code ✅ Funciona
⌘ ⇧ ⌥ U
Encurta a URL usando TinyURL e automaticamente abre um QR Code da URL encurtada no navegador.
Ideal para: Compartilhar links rapidamente via QR Code
🔗 Bit.ly ⚠️ Precisa Token
⌘ ⇧ ⌃ U
Encurta URLs usando o serviço Bit.ly. Requer configuração de token API no código.
Configuração: Adicione seu token Bit.ly na variável BITLY_TOKEN

⚙️ Como Usar Este Projeto

  1. Instale o Hammerspoon: brew install --cask hammerspoon
  2. Clone o repositório: git clone https://github.com/miranda-rodrigo/macOS-useful-automations-hammerspoon.git
  3. Copie o arquivo: cp init.lua ~/.hammerspoon/init.lua
  4. Recarregue a configuração: Abra o Hammerspoon e clique em "Reload Config"
  5. Configure permissões: Vá em Configurações → Privacidade e Segurança → Acessibilidade e adicione o Hammerspoon
  6. Teste os 12 atalhos! 🎉
